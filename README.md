# CSharpDatabaseApplication
The goal here is to create a simulated POS system for a grocery store.
The POS system allows the user to add and remove items, find their costs, and determine things about value.
Cost functions have been added.
Login system has been added and now has its own table in the database.
Login permissions have been added and now some of these permissions are active.
